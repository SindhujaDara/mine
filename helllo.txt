              setState(() {

              });
            },
            child: Text("ADD"),
          ),
          Text(result, style: TextStyle(fontSize: 25)),
        ],
      ),
    );
  }

2.a) 
 **1. Text Widget**

* Displays simple text on the screen.
* Most commonly used widget.
* Supports styling like font size, color, weight.

## **2. Container Widget**

* A box model widget.
* Used for padding, margin, borders, background color, height, width.
* Mainly used for UI design.

## **3. Row Widget**

* Displays children **horizontally** in a row.
* Uses `mainAxisAlignment` & `crossAxisAlignment` for alignment.

## **4. Column Widget**

* Displays children **vertically**.
* Very similar to Row but vertical direction.

## **5. Image Widget**

* Shows images from assets, network, memory, or files.
* Commonly used: `Image.asset()` or `Image.network()`.

## **6. Icon Widget**

* Displays built-in icons from Material Icons library.
* Can set size and color.

## **7. Scaffold Widget**

* Provides basic visual layout structure.
* Contains AppBar, Body, FloatingActionButton, Drawer, etc.

## **8. AppBar Widget**

* A horizontal bar at the top.
* Contains title, actions (buttons), and background color.

## **9. ElevatedButton Widget**

* A Material Design button.
* Used for actions like submit, click, next.
* Has elevation and background color.

## **10. TextField Widget**

* Used to take input from the user.
* Supports controllers, keyboard type, decoration, validation.

## **11. ListView Widget**

* Scrollable list of widgets.
* Can be vertical or horizontal.
* Used when showing long lists (contacts, messages, products).

## **12. Stack Widget**

* Places widgets **on top of each other**.
* Used for overlapping UI elements.

## **13. Card Widget**

* Displays content inside a Material Design card.
* Has rounded corners, shadows.

## **14. SizedBox Widget**

* Used to create space between widgets.
* Can also set fixed width or height.

## **15. Padding Widget**

* Adds internal spacing around a child widget.

## **16. Center Widget**

* Centers the child inside the available space.

## **17. AlertDialog Widget**

* Displays pop-up dialog boxes.
* Used for warnings, confirmations, messages.

## **18. Drawer Widget**

* A side navigation panel that slides from the left.

## **19. BottomNavigationBar Widget**

* A bar at the bottom with multiple tabs.
* Used for navigation between screens.

## **20. GridView Widget**

* Displays items in a grid (rows and columns).
* Used for gallery view, product listing, dashboards.

////////////RESPONSIVE UI TO DIFFERENT SCREENS //////////////////
import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text("Ultra Simple Responsive UI")),
        body: LayoutBuilder(
          builder: (context, constraints) {
            // if width < 600 = mobile
            if (constraints.maxWidth < 600) {
              return Column(
                children: [
                  Container(height: 100, color: Colors.red, margin: EdgeInsets.all(10)),
                  Container(height: 100, color: Colors.blue, margin: EdgeInsets.all(10)),
                ],
              );
            } else {
              // tablet or bigger
              return Row(
                children: [
                  Expanded(child: Container(height: 100, color: Colors.red, margin: EdgeInsets.all(10))),
                  Expanded(child: Container(height: 100, color: Colors.blue, margin: EdgeInsets.all(10))),
                ],
              );
            }
          },
        ),
      ),
    );
  }
}

/////////////// MEDIA QUERIES AND BREAK POINTS //////////////////////////////

import 'package:flutter/material.dart';

void main() => runApp(MyApp());

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text("MediaQuery Example")),
        body: ResponsiveText(),
      ),
    );
  }
}

class ResponsiveText extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    double width = MediaQuery.of(context).size.width;  // Get screen width
    String view;

    if (width < 600) {
      view = "Mobile View";
    } else if (width < 1200) {
      view = "Tablet View";
    } else {
      view = "Desktop View";
    }

    return Center(
      child: Text(
        view,
        style: TextStyle(
          fontSize: width < 600 ? 20 : width < 1200 ? 28 : 36,
          fontWeight: FontWeight.bold,
          color: Colors.blue,
        ),
      ),
    );
  }
}


/////////////////////// NAVIGATORS WITH NAMED ROUTES ////////////////////////////

import 'package:flutter/material.dart';

void main() {
  runApp(MaterialApp(
    initialRoute: '/',   // Starting screen
    routes: {
      '/': (context) => Screen1(),      // Name for Screen1
      '/screen2': (context) => Screen2() // Name for Screen2
    },
  ));
}

class Screen1 extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text("Screen 1")),
      body: Center(
        child: ElevatedButton(
          child: Text("Go to Screen 2"),
          onPressed: () {
            Navigator.pushNamed(context, '/screen2'); // Go using route name
          },
        ),
      ),
    );
  }
}

class Screen2 extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text("Screen 2")),
      body: Center(
        child: ElevatedButton(
          child: Text("Back"),
          onPressed: () {
            Navigator.pop(context); // Go back
          },
        ),
      ),
    );
  }
}

import 'package:flutter/material.dart';
import 'package:provider/provider.dart';

void main() {
  runApp(
    ChangeNotifierProvider(
      create: (_) => Counter(),
      child: MaterialApp(home: CounterScreen()),
    ),
  );
}

class Counter with ChangeNotifier {
  int count = 0;
  void inc() {
    count++;
    notifyListeners();
  }
}

class CounterScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    var c = Provider.of<Counter>(context);

    return Scaffold(
      appBar: AppBar(title: Text("Provider")),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            Text("Count: ${c.count}"),
            ElevatedButton(
              onPressed: () {
                c.inc();
              },
              child: Text("Increase"),
            ),
          ],
        ),
      ),
    );
  }
}
